### 关键决策

| 决策 | 依据 |
|------|------|
| MVP 不先做数据库 | 当前客户只需要月度报告，月度 JSON 工件足够支撑交付和复核 |
| 保留数据库设计为第二版 | 后续若需要长期运营、跨月检索、历史重打标，可从 JSON adapter 接入 SQLite |
