# product

这里放最终产品相关内容：产品代码、配置、爬虫、运行输出。

| 目录 | 作用 |
|------|------|
| `croda-frontend/` | 前端代码和报告/管理界面 |
| `croda-backend/` | 后端 API、任务编排、数据服务 |
| `skills/` | 客户交付的 Xiaolongxia/OpenClaw skills，例如打标、报告生成等产品流程 skill |
| `spider/` | 微信和网站爬虫代码 |
| `parameter-file/` | Watchlist、source profiles、prompt、validator 等配置 |
| `output/` | 唯一运行输出根目录 |

不要把项目管理文档、设计记录、工作日志或内部项目管理 Agent skill 放在这里。客户需要安装或在产品流程中读取的 skill 应放在 `skills/`。
