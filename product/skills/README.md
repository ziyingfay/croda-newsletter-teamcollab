# Product Skills

This directory contains customer-deliverable Xiaolongxia/OpenClaw skills used by the Croda newsletter product workflow.

Put skills here when the customer is expected to install them or when the production workflow reads them directly.

Current skills:

| Skill | Purpose |
|-------|---------|
| `newsletter-tagging/` | Xiaolongxia 1 article tagging skill for Croda Beauty market-intelligence articles |

Internal project-management, repository-organization, collaboration, or governance skills belong under `project-management/skills/`, not here.
