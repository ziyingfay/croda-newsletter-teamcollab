# project-management

This area contains project-management assets, references, and internal agent management skills, not final product code.

| Directory | Purpose |
|-----------|---------|
| `reference/` | User-provided requirements, source material, examples, and research |
| `skills/` | Internal project-management, repository-organization, collaboration, and governance skills |

Development logs, requirement-change notes, current-state files, temporary scratch work, and skill-development records are intentionally not included in this clean template.

Actual product code, customer-deliverable skills, configuration, and runtime outputs live in `../product/`.
