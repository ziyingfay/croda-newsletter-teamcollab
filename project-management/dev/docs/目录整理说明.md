后续不要再新建根级 `app/`、`outputs/`、`dev/`、`reference/`、`newsletter-tagging/`。对应内容分别进入：

- 产品代码：`product/croda-backend/`、`product/croda-frontend/`、`product/spider/`
- 运行输出：`product/output/`
- 管理文档与日志：`project-management/dev/`
- 参考资料：`project-management/reference/`
- 客户交付 / 产品流程 Agent skills：`product/skills/`
- 内部项目管理 Agent skills：`project-management/skills/`
