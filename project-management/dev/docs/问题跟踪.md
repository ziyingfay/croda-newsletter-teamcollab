## 状态定义

| 状态 | 说明 |
|------|------|
| Open | 已发现，尚未处理 |
| In Progress | 正在处理 |
| Resolved | 已修复，待验证 |
| Closed | 已验证关闭 |
| Not A Bug | 调查后确认不是问题 |

## 严重性定义

| 严重性 | 说明 |
|--------|------|
| High | 阻塞核心流程或造成明显错误 |
| Medium | 影响核心能力或阶段交付 |
| Low | 小问题、优化项、文档问题 |
