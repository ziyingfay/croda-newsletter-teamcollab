# reference 目录说明

参考资料区，默认只读。

| 目录 | 用途 |
|------|------|
| `requirements/` | PRD、需求文档、客户说明 |
| `source-materials/` | 原始素材、旧系统、用户提供资料 |
| `research/` | 调研资料、竞品分析、技术参考 |
| `examples/` | 可参考的外部样例、历史样例 |
| `archive/` | 过期但可能还要追溯的资料 |

清理提示：

- `requirements/` 和 `source-materials/` 不要清理。
- 不确定的资料先移入 `archive/`，不要直接删除。
