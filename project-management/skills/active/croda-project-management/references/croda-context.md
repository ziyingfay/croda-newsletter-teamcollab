# Croda Context Summary

Croda / 禾大 Beauty is a B2B beauty and personal-care ingredients supplier. The project goal is a monthly intelligence newsletter workflow that turns RSS, WeChat, and website inputs into auditable JSON artifacts and an interactive HTML report.

## MVP Flow

```text
RSS / 公众号抓取
→ 干净 RSS JSON
→ 小龙虾 1 打标 JSON
→ 小龙虾 2 报告内容 JSON
→ HTML 渲染脚本
→ 最终月报
```

MVP should use monthly folders and JSON files. Keep the database design as a future second-version option, not an MVP blocker.

## Monitoring Scope

Monitor:

- International ingredient competitors such as BASF, Symrise, Evonik, Wacker, DSM-Firmenich, Ashland, Lubrizol, Givaudan, Clariant / Lucas Meyer, and Syensqo.
- Domestic challengers and new ingredient suppliers.
- Distributors and agents.
- Downstream MNC and China beauty brands.
- Beauty media, events, databases, and regulatory sources.
- Croda's own product and media signals, especially Matrixyl and peptide-related topics.

## Newsletter Themes

Prioritize events and insights around ingredients, formulas, new products, technologies, innovation, regulation, market activity, customer needs, and Croda-relevant business opportunity.

Common topics include peptides, PDRN, ceramides, retinol alternatives, hyaluronic acid, probiotics / microbiome, delivery systems, AI R&D, synthetic biology, blue biotechnology, sustainability, EU regulation, exhibitions, and customer quality trends.
